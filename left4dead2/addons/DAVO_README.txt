

Don't subscribe to mods in Steam because slows down startup

Mods 'outside' of the workshop folder load instantly

To find mod pages, put ID in place of X:
https://steamcommunity.com/sharedfiles/filedetails/?id=X

Use "open_links.bat" to get links from workshop folder

Download mods using:
https://steamworkshopdownloader.io/

VPK FILENAME FORMAT:
<part of game it changes>_<descriptor/theme/character>_<ID>
